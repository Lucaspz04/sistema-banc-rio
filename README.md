# Sistema Bancário em Python

Projeto simples de sistema bancário feito em Python para praticar lógica de programação.

## Funcionalidades

- Cadastro de usuário
- Login
- Consulta de saldo
- Depósito
- Saque

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos básicos de programação, como:

- dicionários
- estruturas de repetição
- condicionais
- organização de lógica

## Como executar

1. Baixe o arquivo do projeto
2. Abra o terminal na pasta
3. Execute o comando:

python sistema_bancario.py
